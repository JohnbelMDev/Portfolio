Designer Nail Salon

Project Description

A fully responsive nail salon website. This website includes a main page, contacts page, services page, and a reviews page. Integrated the JavaMail API to implement a Simple Mail Transfers Protocol (SMTP) to send out appoinment forms submitted by customers to the business email. Used Heroku, a cloud hosting platform for deployment of website.

Project Stack

  • HTML
  • CSS
  • Bootstrap
  • Java
  • Servlets
  • JavaMail API
  • Heroku
